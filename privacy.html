隐私政策

更新日期：2025年07月01日
小账本

武汉捷运网络科技有限公司 (简称"我们")高度重视个人信息保护，我们按照法律法规要求，并参照行业最佳实践，充分保障用户的个人信息安全。鉴于此，我们制定本隐私政策，以帮助用户充分了解个人信息的处理情况。

我们收集的信息范围
1. 用户主动提交的信息

用户主动提交的个人信息。个人信息是指以电子或者其他方式记录的、能够单独或者与其他信息结合识别自然人身份或反应特定自然人活动的各种信息。除非用户主动提交或取得用户的同意，我们不会擅自收集用户的个人信息。

用户主动提供的非个人信息。在用户使用我们的产品与服务过程中，视功能需要用户也可能会向我们主动提供部分非个人信息。这些非个人信息一般无法识别您的个人身份。例如，在使用我们的翻译棒硬件产品时，您可能会向我们提交您的具体语音指令，以便我们能够识别解析用户的语音内容，向用户提供翻译服务，同时收集这些语音内容也会协助我们了解用户的发音习惯，提高翻译的准确度。

2. 自动收集的信息

为了分析用户的使用习惯、改善用户体验和优化我们产品与服务的性能，我们会自动从您访问应用程序的手机、平板电脑以及其他设备（以下简称"设备"）收集某些日志、行为、设备信息以及诊断信息等。这些信息包括：

2.1 日志及设备信息

收集方式：

指您使用我们的服务时，在获取您设备信息权限后，系统可能通过Cookie或其他方式自动采集的日志信息，在您使用产品过程中（含静默状态）收集设备信息判断账号是否存在异常以便及时响应网络异常问题。

应用在后台状态下获取您的设备标识（如IMEI/IMSI/AndroidID/IDFA/OPENUDID/GUID、MAC）为维护我们服务的正常运行，排查产品故障，我们有可能会获取您的设备标识，这类信息是为提供服务必须收集的信息，收集后即作去标识化处理，不会用于识别任何特定的用户。

收集范围：

  • 设备或软件信息，例如您的移动设备、网页浏览器或用于接入我们服务的其他程序所提供的设备型号、MAC地址、IMEI、IMSI、OAID、AndroidID、配置信息、您的IP地址和移动设备所用的版本和设备唯一性识别码；
  • 接入的三方广告SDK（穿山甲、优量汇、快手、百度等）会在后台运行服务，不定期获取设备的信息（MAC地址、IP、IMEI、IMSI、AndroidID、OAID、和软件安装列表）。收集的数据会进行匿名化存储，且不会用于其他目的。
  • 在使用我们服务时搜索或浏览的信息，例如您使用的网页搜索词语、访问的社交媒体页面url地址，停留时长、刷新记录、发布记录、关注、订阅、收藏、分享以及您在使用我们服务时浏览或要求提供的其他信息和内容详情；
  • 有关您曾使用的移动应用（APP）和其他软件的信息，以及您曾经使用该等移动应用和软件的信息；
  • 您通过我们的服务分享的内容所包含的信息（元数据），例如拍摄或上传的共享照片或录像的日期、时间或地点。

收集目的：以识别您的帐号异常状态、了解产品适配性，保障我们服务的功能、网络及运营安全，并向您提供更契合您需求的页面展示、浏览、搜索服务。

2.2 位置信息（敏感个人信息）

收集方式：在您开启位置权限后，我们可以获取您的位置信息。

收集范围：

  • 您通过具有定位功能的移动设备使用我们的服务时，我们会不定时的通过GPS等方式收集的您的精准地理位置信息；
  • 您或其他用户提供的包含您所处地理位置的实时信息，例如您提供的账户信息中包含的您所在地区信息，您或其他人上传的显示您当前或曾经所处地理位置的共享信息，您或其他人共享的照片包含的地理标记信息。

收集目的：以便我们为您提供优化服务，例如根据位置信息提供更符合您的功能需求。

2.3 软件列表信息

收集方式：软件列表信息（包括当前已安装应用软件列表、当前或最近运行的软件列表和运行中的进程），指您使用我们软件相关功能时，收集的软件信息，主要用于帮助您恢复已备份软件时，避免重复下发本地已有软件。

收集范围：应用列表的软件安装、使用记录，文件下载、缓存信息。

收集目的：我们将访问您的存储空间，以便使您可以下载、保存、清理内容，或者通过缓存来实现加速优化服务等功能，并用于帮助您恢复已备份软件时，避免重复下发本地已有软件；检测软件安装情况，便于完成手机加速、软件及文件清理功能。

2.4 其他基于单项服务需要您在您的设备中开启特定的访问权限

下列权限均不会默认开启，除非获得您的明确授权，如您不开启下列授权，亦不会影响您正常体验我们的基本功能，例如：

  • 在您开启通讯录权限后，我们可以获取您的通讯录好友信息，以便向您提供好友状态查询、邀请您的好友使用我们的服务、优化您的通讯录、向您认识的人分享内容/产品等功能；
  • 在您开启我们可读取/写入您日历的权限后，我们可以获取您的日程时间，以便为您提供记录、提醒等功能；
  • 在您开启我们通知栏使用的权限后，我们可以获取您通知栏内容以便向您提供清理通知内容的服务，此外还可向您提供信息提示的功能，以便您更快捷进入我们的产品；
  • 读取剪切板内容，以便向您提供复制、粘贴功能。
2.5 收集陀螺仪传感器、加速度传感器、重力传感器

为了适配您的设备状态，我们会调用设备的重力、加速度等传感器，以便适配您的设备横竖屏状态。

对于比较敏感的用户信息，如用户IMEI、IMSI、GPS位置信息等，我们在收集之前会征得您的同意。对于一些非敏感信息的收集，则可能是默认和自动的。您可以通过调整移动设备的"设置"选项限制我们从您的移动设备收集相关信息或使用特定权限。当您限制我们获取某项权限后，可能会影响某项功能的运行，导致我们可能无法继续为您提供相对应的服务，也不再处理您相应的信息。

2.6 我们会获取您安卓设备的部分权限

我们主要会获取的权限以及对应的用途如下：

  • 获取存储权限：用于图片视频恢复、语音导出；
  • 获取手机状态权限：获取设备识别码用于独立设备标记以及第三方广告SDK展示广告，包括穿山甲、优量汇、快手、百度广告SDK等；
  • 获取运行列表权限：用于软件管理、卸载应用等功能；
  • 访问WLAN、蜂窝移动网络：用于本产品与服务器时正常的数据同步与数据交换；
  • 获取Camera相机：用于本产品内放大器功能，拍摄图像，并支持缩放；
  • 获取SD卡数据：用于本产品清理功能扫描SD卡内容。
2.7 收集 WiFi 相关的信息

收集的 WiFi信息范畴：

  • SSID（服务集标识符）：也就是 WiFi 网络的名称；
  • BSSID（基本服务集标识符）：代表 WiFi 接入点的 MAC 地址；
  • 信号强度：通过 RSSI 值（dBm）来体现；
  • 加密类型：例如 WPA2、WPA3 等；
  • 其他技术参数：像信道、频率、网络类型（2.4G/5G）等；

信息收集的目的：

  • 优化网络连接：借助分析周围 WiFi 的信号状况，为用户推荐最为优质的网络连接选项；
  • 提升服务质量：用于检测网络覆盖的薄弱区域，对服务器的部署位置进行优化；
  • 安全检测：识别潜在的钓鱼 WiFi 或风险网络；
  • 设备定位：协助辅助定位；

信息收集的方式：

  • 设备权限获取：在用户授权的情况下，通过访问设备的 WiFi 扫描接口来获取信息；
  • 非侵入式收集：不会主动连接未授权的 WiFi 网络，仅对可见的网络信息进行读取；
  • 匿名化处理：在进行数据统计分析时，会移除用户的个人身份标识；
2.8 三方 SDK 信息收集说明

我们的应用集成了抖音、快手、广点通、百度、高德、友盟等三方 SDK（具体见如下），在运行过程中，上述 SDK 存在收集设备 MAC、Android ID、OAID、MEID、WiFi、SSID、BSSID、ICCID、SN、SUCI、SUPI、软件安装列表、位置、联系人、通话记录、日历、 短信、本机电话号码、图片、音视频、传感器数据、注册传感器监听器的行为，具体说明如下：

  • 收集目的：三方 SDK 收集设备信息，主要用于实现产品功能或广告的优化，提升功能的精准度、分析用户使用场景与行为偏好，以提供更贴合您需求的功能或广告内容，同时保障广告投放的安全性和有效性；
  • 收集方式：通过在您使用应用过程中，当触发相关产品功能、广告展示或交互场景时，SDK自动启动并收集设备数据;部分 SDK 会在您使用过程中不定时的收集一些设备信息，如设备位置、设备传感器数据等。
  • 收集范围：仅收集与产品功能、广告投放相关的设备信息、传感器数据，包括但不限于设备运动状态数据、设备方向数据等，具体以 SDK实际获取内容为准；
